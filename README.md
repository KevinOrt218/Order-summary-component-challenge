# Order Summary Component Challenge

## Descripción
Reto de Frontend Mentor: una tarjeta de resumen de pedido con imagen, detalle del plan, precio y dos acciones ("Proceed to Payment" y "Cancel Order").

## Qué practiqué
- **Flexbox** para centrar verticalmente y horizontalmente un componente único en el viewport.
- Estilado de botones y enlaces con estados `:hover` / `:focus` para mejorar la interacción.
- Uso de imágenes de fondo/ilustraciones (pattern) combinadas con el layout de la tarjeta.
- Buenas prácticas de espaciado consistente usando `rem` y una escala tipográfica definida.

## Tecnologías
HTML5, CSS3

## Aprendizaje clave
Practiqué a construir un componente "card" completo con múltiples elementos internos (imagen, texto, precio, botones) manteniendo jerarquía visual clara y buen espaciado.
